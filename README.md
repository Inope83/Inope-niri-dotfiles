# 🪟 Niri Dotfiles

> Minimal, modern, and dynamic Niri Wayland compositor configuration for Fedora

## ✨ Features

- 🎨 **Dynamic Colors** - Automatic theming with Pywal
- 🖥️ **Modern Status Bar** - Waybar with Catppuccin theme
- ⌨️ **Comprehensive Keybindings** - Visual cheatsheet with Mod+Shift+B
- 🖼️ **Wallpaper Management** - Random wallpaper with smooth transitions
- 🔄 **Auto-reload** - Changes apply instantly

## 🚀 Quick Install

```bash
git clone https://github.com/Inope83/INOPE-Niri-Dotfiles
cd niri-dotfiles
./install.sh
